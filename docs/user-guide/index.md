# User Documentation

This section covers day-to-day usage of ChurchCRM for end users. For system administration tasks, see the [Administration](Administration) section.

---

## 🚀 Quick Start Guide

New to ChurchCRM? Here's how to get started:

1. **Add your first family** → [Families](Families)
2. **Add family members** → [Persons](Persons)
3. **Organize into groups** → [Groups](Groups)
4. **Set up classifications** → [Classifications](Classifications)
5. **Start tracking donations** → [Finances](Finances)

> **Tip:** Use the [Search](Search) feature (press `?`) to quickly find any person, family, or group.
