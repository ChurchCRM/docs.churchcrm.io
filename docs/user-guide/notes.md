# Notes

Notes are miscellaneous memos that can be attached to [Person](Persons) or [Family](Families) records. Use them to record pastoral visits, prayer requests, important conversations, or any other relevant information.

## Note Privacy

| Type | Who Can See |
|------|-------------|
| **Public** | All users with Notes permission |
| **Private** | Only the user who created the note |

> **Tip:** Use private notes for sensitive pastoral information that shouldn't be shared with all staff members.

---

## Viewing Notes

1. Open the [Person](Persons) or [Family](Families) record
2. Scroll to the **Notes** section at the bottom
3. Notes are displayed in reverse chronological order (newest first)

---

## Adding a Note

1. Open the Person or Family record
2. In the **Notes** section, click **Add a Note to this Record**
3. Enter your note text (no length limit)
4. Check **Private** if only you should see this note
5. Click **Save**

## Editing a Note

1. Find the note in the Notes section
2. Click **Edit this Note**
3. Make your changes
4. Click **Save**

> **Tip:** To make a private note public, edit it and uncheck the **Private** checkbox.

---

## Deleting a Note

1. Find the note in the Notes section
2. Click **Delete this Note**
3. Confirm the deletion

> **Warning:** Deleted notes cannot be recovered.
