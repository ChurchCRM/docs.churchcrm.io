# Volunteers

The Volunteers feature helps you track member talents and willingness to serve in specific roles. This makes it easy to find the right people when needs arise.

## Common Volunteer Opportunities
- Baking / Meal preparation
- Childcare / Babysitting
- Grounds maintenance
- Music ministry
- Ushering / Greeting
- Technical support
- Teaching / Tutoring

## Setting Up Volunteer Opportunities

1. Navigate to **Admin → Volunteer Opportunity Editor**
2. Add the volunteer opportunities relevant to your church
3. For each opportunity, you can specify a description

## Assigning Volunteers to Opportunities

1. Open a [Person's](Persons) record
2. Look for the "Volunteer Opportunities" section
3. Check the opportunities they're willing to help with
4. Save the changes

## Finding Volunteers

When you need volunteers for a specific task:

1. Navigate to **Data/Reports → Query Menu**
2. Scroll to the bottom and click **Volunteers**
3. Select the volunteer opportunity you need help with
4. The query will return all [people](Persons) who volunteered for that task
5. Add results to the [Cart](Cart) for further actions:
   - Create a contact directory
   - Print address labels
   - Send emails
